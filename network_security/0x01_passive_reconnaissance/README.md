# 0x01 Passive Reconnaissance

## Overview
This module covers passive reconnaissance techniques used in network security assessments. Passive reconnaissance involves gathering information about a target without direct interaction.

## Learning Objectives
- Understand passive information gathering methodologies
- Master OSINT (Open Source Intelligence) techniques
- Learn to identify and analyze publicly available data
- Develop reconnaissance skills for security assessments

## Topics Covered
- WHOIS lookups and domain registration information
- DNS enumeration and analysis
- Search engine reconnaissance
- Social media and public data mining
- Network topology discovery
- Service fingerprinting
