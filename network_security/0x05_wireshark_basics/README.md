# Network Security - Wireshark Basics

Ce répertoire contient des solutions et des explications pour le projet "Wireshark Basics" dans le cadre du module Network Security. L'objectif est de se familiariser avec l'outil d'analyse de paquets Wireshark, comprendre le flux de trafic réseau et analyser des captures `.pcap`.

## Ressources

*   [Site Officiel de Wireshark](https://www.wireshark.org/)
*   [Documentation Wireshark](https://www.wireshark.org/docs/)
*   [Sample Captures (Wiki Wireshark)](https://wiki.wireshark.org/SampleCaptures)

## Prérequis

*   Wireshark installé sur votre machine (Windows, macOS ou Linux).
*   Compréhension de base des modèles OSI et TCP/IP.

## Objectifs d'apprentissage

À la fin de ce projet, vous devriez être capable d'expliquer :

*   Qu'est-ce que Wireshark et à quoi il sert.
*   Comment capturer du trafic réseau.
*   Comment filtrer les paquets (filtres d'affichage et filtres de capture).
*   Comment analyser les en-têtes de paquets (Ethernet, IP, TCP, UDP, etc.).
*   Comment suivre un flux TCP ou UDP.

## Structure du projet

Les tâches de ce projet impliquent généralement l'analyse de fichiers `.pcap` fournis et la réponse à des questions spécifiques basées sur les données contenues dans ces captures.
