# Holberton School - Cyber Security

## Description

This repository contains projects and exercises for the Cyber Security curriculum at Holberton School. It covers various aspects of cybersecurity including network security, ethical hacking, penetration testing, and security best practices.

## Learning Objectives

- Understand fundamental concepts of cybersecurity
- Learn about network security and protocols
- Practice ethical hacking techniques
- Develop skills in penetration testing
- Implement security best practices

## Requirements

- All scripts will be tested on Ubuntu 20.04 LTS
- All scripts should end with a new line
- A `README.md` file at the root of the folder is mandatory
- All Bash script files must be executable
- Follow security and ethical guidelines at all times

## Repository Structure

```
holbertonschool-cyber_security/
├── README.md
└── [project_folders]/
```

## Usage

Each project folder contains its own README with specific instructions and requirements.
